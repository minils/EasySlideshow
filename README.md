### Compile from source

To compile from source you need to have the following dependencies installed:
- Qt5
- qmake

```sh
git clone https://github.com/minils/EasySlideshow.git
cd EasySlideshow
mkdir build
cd build
export CONFIG=release
qmake ../
make
```
